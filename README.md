# AirCal - Clarity in Every Breath

This is the official web application for AirCal, an open-source platform for real-time, air quality monitoring. Our mission is to make air quality data transparent, accessible, and actionable for everyone.

## Key Features
- **Open-Source Platform**: Both our hardware schematics and software are open-source, promoting transparency, collaboration, and community-driven innovation.
- **Community-Driven Data**: Data from every sensor in the network contributes to a global, open-source map, creating a powerful resource for public health and environmental science.
- **Real-Time Map**: A live, interactive map (coming soon) will visualize air quality data from around the world.

## Technology Stack

This project is built with a modern, performant, and scalable technology stack:

- **Framework**: [Next.js 14](https://nextjs.org/) (with App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **UI**: [React](https://react.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You need to have [Node.js](https://nodejs.org/) (version 18 or later) and npm installed on your machine.

### Installation

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/Air-Calibre/AirCalibre.git
    cd AirCal-Demo-Website
    ```

2.  **Install NPM packages:**
    ```sh
    npm install
    ```

3.  **Run the development server:**
    ```sh
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
